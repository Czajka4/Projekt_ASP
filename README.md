# flask-asp

### GUI - na porcie 8000 wyświetla wyniki
### API - na porcie 5000 nasłuchuje w oczekiwaniu request od GUI
#
###  Budowanie projektu
```
 docker-compose build 
```
 #
 ### Uruchamianie projektu
 ```
 docker-compose up -d
 ```
  #
 ### Sprawdzanie logów
 ```
 docker-compose logs
 ```
  #
 ### Link do strony
 http://localhost:8000/stock
